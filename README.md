# FightGame

Developed with Unreal Engine 5

Movements

      Light Attack
              Keyboard: U
              Xbox Controller: X

      Medium Attack
              Keyboard: I
              Xbox Controller: Y

      Hard Attack
              Keyboard: O
              Xbox Controller: B
        


Combos

      Archer Combo: Light Attack, Back, Light Attack
      Knight Combo: Light Attack, Front, Hard Attack



Commands

      ChangeBoxesVisibility : Change the visibility of hitboxes and capsules
